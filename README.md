## Getting Started

## Installation

1. Clone the repo
   ```sh
   git clone git@github.com:shavinski/personal-portfolio.git
   ```
2. Install dependencies
   ```sh
   npm install
   ```
3. Run your local server
   ```sh
   npm start
   ```
